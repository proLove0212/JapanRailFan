# JapanRailFan - Japanese Rail Information


## About
A website for viewing different types of Japanese train rolling stock, lines, operators and tickets. It contains information from a small selection of lines in Tokyo and Osaka.

## Features
- View train, line, operator and ticket information (and information related to it)
- View ticket valid lines
- Ticket calculator with save/load/import/export options
- Easy to include additional information via data.js

## How to expand the collection
You can add additional information via data.js by dupilcating the object structure.
